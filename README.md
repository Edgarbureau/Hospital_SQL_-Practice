# Hospital_SQL_-Practice
SQL practice using hospital database, covering joins, CTEs, window functions, subqueries, aggregate, date, and string functions.
OVERVIEW
This repository contains SQL practice queries that was written using a sample hospital database. it shows the core SQL skills used in data analysis and healthcare analytics.
SKILLS DEMONSTRATED INCLUDE;
SELECT statements, WHERE, GROUP BY, HAVING, ORDER BY, Aggregate functions(SUM, COUNT, AVG, MIN, MAX), INNER JOIN, LEFT JOIN, SELF JOIN, Subqueries, CTEs, Window Functions (ROW_NUMBER(), DENSE_RANK(), LAG(), Running Totals, Date Functions, String Functions.
PURPOSE
To strengthen my SQL skills by solving analytical queries related to my niche using healthcare data.
